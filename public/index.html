The Guest List is a submission for the Chart.js Personal Dashboard Challenge. This personal dashboard visualizes the guest list for our wedding! You can see the supporting data under the "Guests" tab. The dashboard isn't limited just to a guest list though - you can use any data you want. You only need two arrays of JSON objects: one for all your data (eg. an array of guest objects) and one that describes your data. Check out the source code for details on how this works.

This dashboard is hosted by Heroku and the source code is available on Github.

Credit to Chart.js for awesome charts and Bootstrap for awesome CSS!
